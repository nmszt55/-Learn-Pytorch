### 安装组件

注：如果您已经编译安装完成，只需重新编译安装一遍即可

安装Tkinter支持：

Python2:

```bash
sudo apt-get install python-tk	
```

Python3:

```Bash
sudo apt-get install python-tk
```

安装LZMA支持：

```Bash
apt-get install liblzma-dev -y
```

注：tkinter如果安装后还有问题，可能是因为Python的版本太高，建议选择较为稳定的版本
